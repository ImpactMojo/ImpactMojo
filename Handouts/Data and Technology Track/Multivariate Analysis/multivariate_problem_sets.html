Answer Key & Hints

Problem Set 1 Hints:

  • Part A: Remember correlation strength categories: weak (< 0.3), moderate (0.3-0.7), strong (> 0.7)
  • Part B: F-statistic > critical value means reject null hypothesis of equal means
  • Part C: For prediction, substitute values into regression equation: Y = 52.34 + 1.24(5) + (-0.31)(60) + (-0.89)(10) + (-0.15)(80) = 21.64%

Problem Set 2 Hints:

  • Part A: One-way ANOVA is appropriate for comparing means across multiple groups
  • Part B: Effect size (Cohen's d) > 0.8 is considered large; here it's about 1.9 (very large)
  • Part C: Consider both statistical significance and practical significance when building models

Problem Set 3 Hints:

  • Sequential approach: Start with correlations to identify key relationships, use ANOVA for categorical comparisons, build regression for prediction
  • Policy focus: Transparency and governance quality often matter more than just funding
  • Equity analysis: Look at gaps between overall access and access for disadvantaged groups
