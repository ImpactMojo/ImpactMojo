Professional
₹999/month · ₹9,990/year (2 months free)

Includes everything in Practitioner, plus:

Qualitative Research Lab

AI-assisted qualitative analysis for interview and FGD transcripts. Upload transcripts and get thematic coding suggestions, memo generation, pattern identification, and cross-case comparisons. Supports iterative coding with human-in-the-loop validation. Built for development researchers who need rigorous qual analysis without NVivo licensing costs.

Statistical Analysis Assistant

Interactive tools for regression diagnostics, effect size calculation, and statistical power analysis. Covers OLS, logistic regression, difference-in-differences, regression discontinuity, and instrumental variables estimation. Input your parameters and get sample size requirements, minimum detectable effects, and diagnostic plots.

Statistical Code Convertor

Translate analysis scripts between Stata, R, Python, and SPSS instantly. Paste your do-file and get equivalent R or Python code with package recommendations. Particularly useful for teams transitioning between statistical software or collaborating across platforms.

VaniScribe AI Transcription

Transcribe field interviews and focus group discussions in Hindi, Tamil, Bengali, Marathi, Telugu, Kannada, Malayalam, Gujarati, Odia, Punjabi, and more South Asian languages. Speaker diarization separates voices automatically. Export clean transcripts ready for qualitative coding.

DevData Practice

36 realistic dataset generators producing 840,000+ rows modeled on the structure of DHS, NFHS, ASER, MGNREGA, and other major development surveys. Practice cleaning, merging, and analyzing data without confidentiality concerns. Each generator includes a data dictionary and suggested analysis exercises.

Visualization Cookbook

14 chart types with question-driven recipes. Start with what you want to show ("How has enrollment changed over time?") and get production-ready code in R (ggplot2), Python (matplotlib/seaborn), and Excel. Each recipe includes design rationale, accessibility guidance, and export options for reports and presentations.

DevEconomics Toolkit

11 interactive Shiny apps covering core development economics methods: RCT power calculator, DiD event study simulator, RDD bandwidth selector, synthetic control builder, Foster-Greer-Thorbecke (FGT) poverty measures, Theil/MPI inequality decomposition, cost-effectiveness analysis, benefit incidence analysis, and more. Each app includes worked examples from South Asian datasets.

Priority Coaching & Early Access

Direct coaching access to Varna and Vandana for research design questions, career guidance, and methodology support. Plus early access to every new tool, course, and feature before public release.
