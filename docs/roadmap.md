# Roadmap

Planned features and improvements for ImpactMojo. This is a living document — suggest ideas in [GitHub Discussions](https://github.com/Varnasr/ImpactMojo/discussions).

## In Progress

- [ ] Professional translations of course content in Hindi, Tamil, Bengali, Marathi ([#29](https://github.com/Varnasr/ImpactMojo/issues/29))
- [ ] Cohort-based learning with scheduled deadlines ([#144](https://github.com/Varnasr/ImpactMojo/issues/144))
- [ ] Email notifications and streak reminders ([#145](https://github.com/Varnasr/ImpactMojo/issues/145))

## Planned — Q2 2026

### Platform
- [ ] Analytics dashboard v2 — learner completion funnels, time-on-task, assessment scores
- [ ] Enhanced PWA with push notifications and background sync
- [ ] Razorpay payment integration for premium tiers

### Content
- [ ] Additional foundational courses
- [ ] Video walkthroughs for interactive labs
- [ ] Practitioner interview series (case study format)
- [ ] Regional case study packs (state-level India data)

### Tools
- [ ] Sample Size Calculator (standalone lab)
- [ ] Budget Template Generator
- [ ] Survey Instrument Library

### Community
- [ ] Peer review system for lab outputs
- [ ] Alumni network directory

## Planned — Q3-Q4 2026

- [ ] Certificate marketplace — employer-facing verification portal
- [ ] API for partners — REST API for organizations to integrate ImpactMojo content
- [ ] Full WCAG 2.1 AA accessibility audit and fixes
- [ ] Community-contributed courses — allow verified practitioners to publish
- [ ] Live workshops integration with calendar booking
- [ ] Impact measurement dashboard — track real-world outcomes

## How to Suggest Features

1. Check [existing discussions](https://github.com/Varnasr/ImpactMojo/discussions/categories/ideas) first
2. Open a new discussion in the **Ideas** category
3. Describe the problem you're solving, not just the solution
4. Include examples from your development practice if possible
