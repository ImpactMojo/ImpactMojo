# Qual Insights Guide

## What Is Qual Insights?

Qual Insights (also called Qual Lab Pro) is ImpactMojo's **AI-assisted qualitative analysis tool**. It helps researchers code, organise, and analyse qualitative data — interview transcripts, focus group discussions, open-ended survey responses, and field notes.

If you've ever stared at 50 pages of interview transcripts and wondered where to start, Qual Insights provides structure and AI assistance to speed up the analysis process.

## What Is Qualitative Coding? (For Those New to It)

**Qualitative coding** is the process of reading through text data (like interview transcripts) and labelling sections with "codes" — short descriptions of what that section is about.

**Example:** If an interviewee says, "I can't go to the health centre because the bus only comes once a day and it costs too much," a researcher might code this as:
- **Transport barrier** — the bus frequency
- **Cost barrier** — the travel expense
- **Healthcare access** — the overall theme

After coding all your transcripts, you can see patterns: How many respondents mentioned transport barriers? Which barriers were most common? Did men and women report different barriers?

This process is essential for rigorous qualitative research, but it's labour-intensive. Qual Insights makes it faster without sacrificing rigour.

## What Qual Insights Does

### AI-Suggested Codes
Upload a transcript and Qual Insights suggests initial codes based on the content. You review, modify, accept, or reject each suggestion. This gives you a starting point instead of a blank page.

**Important:** The AI suggests — you decide. Qual Insights does not replace the researcher's judgement. It accelerates the mechanical part of coding so you can spend more time on interpretation.

### Code Management
Organise your codes into a structured codebook:
- Create **code categories** (e.g., Barriers, Facilitators, Outcomes)
- Add **definitions** for each code so your coding is consistent
- Merge or split codes as your analysis evolves
- Track code **frequency** across transcripts

### Thematic Analysis
Once coding is complete, Qual Insights helps you identify themes — higher-level patterns that cut across individual codes. The tool visualises relationships between codes and suggests possible thematic groupings.

### Multi-Transcript Analysis
When you're working with multiple transcripts (which is almost always the case in qualitative research), Qual Insights lets you:
- Code across all transcripts using a consistent codebook
- Compare coding patterns between different respondent groups
- Search across all transcripts for specific codes or text patterns

| **Cost** | Included in Professional tier (₹999/month) | $100–$1,500+ per licence |
| **AI-assisted coding** | Yes — suggests initial codes | Limited or separate add-on |
| **South Asian language support** | Designed for multilingual data | Primarily English-focused |
| **Learning curve** | Designed for practitioners, not full-time researchers | Steep learning curve, complex interface |
| **Runs in browser** | Yes — no installation | Requires desktop installation |
| **Integration with ImpactMojo** | Works alongside VaniScribe (transcription) and courses | Standalone tools |

**Who should use full-featured QDAS software instead?** If you're conducting a large-scale qualitative study with hundreds of transcripts and need advanced features like inter-rater reliability calculations, geographic coding, or video analysis, dedicated QDAS software (NVivo, Atlas.ti, MAXQDA) may be more appropriate. Qual Insights is designed for practitioner-level qualitative analysis — the 5–30 transcript range that most development evaluations involve.

## How Educators Can Use Qual Insights

### Teaching Qualitative Methods
Instead of only explaining coding in theory, give participants access to Qual Insights and a practice transcript. They can experience the coding process hands-on.

### Workshop Exercises
Upload a sample FGD transcript and have participants:
1. Review AI-suggested codes
2. Accept, modify, or reject each suggestion
3. Add their own codes
4. Compare their codebooks with each other
5. Discuss why they coded differently — this discussion is where the deepest learning happens

### For Real Fieldwork
Research teams conducting evaluations can use Qual Insights for their actual analysis. The combination of VaniScribe (transcription) and Qual Insights (analysis) provides an end-to-end qualitative workflow.

## Tips

- **AI suggestions are a starting point, not the answer.** Always review and think critically about suggested codes. The researcher's domain knowledge is irreplaceable.
- **Build your codebook iteratively.** Start with a few broad codes and refine them as you read more transcripts. This is standard qualitative practice — Qual Insights supports it.
- **Combine with VaniScribe.** Transcribe interviews in VaniScribe, then analyse them in Qual Insights. The workflow is designed to be seamless.
- **Code in the language of the data.** If your transcripts are in Hindi, code in Hindi. Forcing everything into English during analysis loses nuance.
- **Qual Insights works best with 5–30 transcripts.** For very large qualitative datasets, consider dedicated QDAS software.
